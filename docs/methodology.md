# Methodology

This document outlines the machine learning methodology used for the Smart Waste Classification project.

## Approach
1. **Exploratory Data Analysis (EDA)**: Understanding data distributions and image characteristics.
2. **Preprocessing**: standardizing image sizes, normalization, data partitioning, and augmentation.
3. **Modeling**: Progressive development starting from a Custom CNN baseline to transfer learning backbones (ResNet50, EfficientNet-B0).
4. **Evaluation**: Comprehensive classification metrics (Accuracy, Precision, Recall, Macro F1, Confusion Matrix).
