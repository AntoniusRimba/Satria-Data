## Tujuan

Memahami karakteristik dataset Big Data Challenge SATRIA DATA 2026 sebelum dilakukan tahap Data Preprocessing dan implementasi Transfer Learning menggunakan ResNet50.

EDA bertujuan menghasilkan pemahaman yang komprehensif mengenai kualitas, struktur, distribusi, dan karakteristik dataset sehingga keputusan pada tahap preprocessing dan pemodelan dapat dilakukan berdasarkan hasil analisis, bukan asumsi.

---

## Instruksi

Ikuti seluruh SOP pada command **/eda**.

Ikuti seluruh aturan proyek yang terdapat pada **CLAUDE.md**.

Gunakan seluruh best practice pada skill **image-eda**.

Pastikan seluruh analisis mengikuti aturan kompetisi **SATRIA DATA 2026**.

## Ruang Lingkup

Fokus analisis meliputi:

- Struktur dataset
- Distribusi kelas
- Jumlah citra setiap kelas
- Resolusi gambar
- Format file
- Visualisasi sampel citra
- Analisis kualitas dataset
- Identifikasi potensi permasalahan data
- Analisis yang mendukung tahap preprocessing berikutnya
